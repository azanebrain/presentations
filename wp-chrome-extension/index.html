Chrome Extension has 3 parts:

  • Backround
  • Dom
  • Popup
* Background: Controls communication between the other two pieces. Retrieves the Posts details and sets content to storage
* DOM: Current page. Gets categories, finds matches, sets Post IDs to storage
* Popup: Populates the template w post content. Shows only the posts that were set by DOM image: https://www.pexels.com/photo/34600/
