- WP has provided a process to take PHP and turn it into JS

The WordPress Way

Enqueueing

wp_enqueue_script(
  'angular',
  'path/to/file.js',
  array('dependency'),
  'vA',
  true
);

Name, path, dependencies, version, in footer

Makes sure assets get loaded once, in the correct order
- Allows versioning
- Watch out for plugins loading the same script

The WordPress Way

Shortcodes

Inline:

[my_ng_app attribute="custom"]

PHP:

do_shortcode( '[my_ng_app attribute="custom"]' )
Shortcode allows you to drop the functionality anywhere
- Attributes let you customize things
endpoint var, api, styling

The WordPress Way

Localize Script

wp_localize_script(
  'my_ng_app',
  'wp_attributes_blob',
  array( 'attr' => $attributes['attribute'] )
);

converts attributes into JS:

/*  */
This is how WP lets us take PHP and make it JS
- It's a global object
